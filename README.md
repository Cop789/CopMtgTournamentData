This is a little project I made just to see what decks are performing well at tournaments.
The main value I am geting here is what decks have a higher day 2 conversion rate and what decks have a good matchup against each other.

I pull data from Melee.gg and using the 2 webscrapers to pull player information and match information.

Currently this only works for Regional Championships and spotlights, will add Pro Tours when I find a good way to filter out the drafts at Pro Tours.

See final webapp at https://copmtgtournamentdata.streamlit.app/
